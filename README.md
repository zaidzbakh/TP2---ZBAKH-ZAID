# Calculatrice Python

## Description du projet
Ce projet est une **calculatrice simple en Python**.  
Elle permet de réaliser les opérations suivantes :  
- Addition  
- Soustraction  
- Multiplication  
- Division  

Le projet est conçu pour fonctionner en **mode console**, avec une interface simple en ligne de commande.

## Méthode de lancement du projet

### Prérequis
- Python installé sur votre machine (version 3.x)  
- Terminal ou invite de commande

### Étapes pour lancer le projet
Cloner le dépôt GitHub sur votre machine et lancer le script :

```bash
git clone https://github.com/username/TP2-NomPrenom.git
Se rendre dans le dossier du projet :

cd TP2-NomPrenom
Lancer le programme :

python calculatrice.py
Suivre les instructions affichées dans le menu pour effectuer les calculs.

Fonctionnalités
Interface en console simple et intuitive

Gestion des erreurs :

Entrée invalide (non numérique)

Division par zéro

Possibilité de quitter le programme proprement
